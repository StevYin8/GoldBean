# 攒金豆 (GoldBean) iOS App

一款专为黄金首饰投资者设计的iOS应用，帮助您记录黄金首饰购买信息，实时追踪价值变化和盈亏情况。

## 功能特性

### 🏆 已实现功能 (v1.0)

- ✅ **黄金记录管理**
  - 添加、编辑、删除黄金首饰记录
  - 记录名称、克重、购买价格、购买日期、备注
  - 支持按总价或单克价格输入

- ✅ **实时价格追踪**
  - 模拟黄金市场价格获取（可配置真实API）
  - 自动计算当前价值和盈亏情况
  - 支持手动刷新价格

- ✅ **投资分析**
  - 总览页面显示投资概况
  - 盈亏金额和百分比计算
  - 彩色标记盈利/亏损状态

- ✅ **数据存储**
  - 使用Core Data本地存储
  - 数据持久化，重启应用不丢失

- ✅ **现代化界面**
  - SwiftUI构建，支持iOS 16+
  - 金色主题设计
  - 响应式布局，适配不同屏幕

## 技术架构

- **前端**: SwiftUI
- **数据存储**: Core Data
- **网络请求**: URLSession (预留API接口)
- **架构模式**: MVVM
- **最低系统要求**: iOS 16.0+

1. **克隆项目**

## 使用说明

### 添加黄金记录
1. 在"记录"页面点击右上角"+"按钮
2. 填写黄金首饰信息
3. 选择按总价或单克价格输入
4. 点击"保存"完成添加

### 查看投资概况
1. 在"总览"页面查看总体投资情况
2. 查看总克重、总投入、当前价值
3. 观察盈亏金额和百分比变化

### 管理记录
1. 点击记录可进入编辑模式
2. 支持修改所有信息
3. 左滑可删除记录

### 设置配置
1. 在"设置"页面配置API密钥
2. 开启/关闭价格更新通知
3. 手动刷新当前金价

## API配置

目前使用模拟数据，要使用真实黄金价格，请：

1. 注册以下API服务之一：
   - [metals-api.com](https://metals-api.com/)
   - [goldapi.io](https://goldapi.io/)

2. 在设置页面输入API密钥

3. 修改`GoldPriceService.swift`中的API调用代码
